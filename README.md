# Treasure-Hunter-Game

A browser-based grid game developed in JavaScript. The game features a treasure hunter controlled by the user, monsters controlled by the computer, treasures, and obstacles on a 10x10 grid. The goal is to collect treasures while avoiding monsters.

## Features

- 10x10 grid with configurable size

- Objects: treasure hunter, monsters, treasures, obstacles

- Turn-based gameplay with user and AI (computer-controlled monsters)

- Dynamic score tracking for user and computer

- Prevents invalid moves and provides error messages

- Game ends automatically when no further moves are possible
## File structure
/treasure-hunter-game

├─ game.html          # Main HTML page

├─ game.js            # Minified game logic (uglified)

├─ game.pretty.js     # Readable version of game logic


## How to use 

- Clone this repository.
  
- Open game.html in any modern browser (Chrome, Firefox, Edge).
